Tags html


Tag <h>

São elementos de cabeçalho, geralmente usados para títulos das páginas, seus valores vão desde do <h1> (maior) até o <h6> (menor).

Tag <p>

A tag pp p é usada em parágrafos, por exemplo em textos ou matérias de um blog

Tag <mark>

A tag mark é usada para dar destaque a uma palavra, um contorno de uma cor diferente.

Tag <small>

Tag usada para deixar alguma palavra ou texto em tamanho menor

Tag <i>

Tag usada para deixar uma palavra ou texto em itálico.

Tag <u>

Tag usada para sublinhar uma palavra ou texto.

Tag <strong>

Tag usada para deixar uma palavra ou texto em negrito.

Tag <ol>

Tag que inicia uma lista, neste caso uma lisa ordenada (ordered list)

Tag <ul>

Tag que inicia uma lista não ordenada (Unordered list)

Tag <a>

Tag que é usada para indicar que há um link referenciado a alguma palavra

Tag <hr>

Tag usada para criar uma linha que separa um texto ou uma sessão da página

Tag <sub>

Tag que é usada para deixar um texto ainda menor que a tag small, porem na parte de baixo em comparação com uma palavra comum.

Tag <sup>

Tag que é usada para deixar um texto ainda menor que a tag small, porem na parte de cima em comparação com uma palavra comum

Tag <blockquote>

Tag usada geralmente para fazer uma citação ao final de um texto, fazendo com que seja atribuido um parágrafo com um espaçamento maior para esta tag.
